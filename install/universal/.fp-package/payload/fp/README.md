# FP Portable Bundle

`SKILL.md` is the canonical router. Hosts infer FP from engineering goals; child skills are internal modules loaded only when the router selects them.

```text
small -> Tiny Brief
unknown cause -> Debug-first
medium -> Execution Brief + Evidence Ledger v1
large/risky -> minimum internal chain -> one compiled brief
```

## Internal Modules

```text
question-requirements/   facts, decisions, failure paths
delete-scope/            seven-rung reuse ladder and MVP cuts
semantic-architecture/   only for real coupling risk
optimize-path/           compile the smallest stable route
shorten-iteration/       evidence-backed micro-loops
evidence-ledger/         canonical completion/failure record
metrics/                 conservative observed metrics
adaptive-improvement/    staged evidence-backed learning
generalization-gate/     blind holdout and negative-control promotion gate
schema-memory/           promoted patterns, not raw context
```
